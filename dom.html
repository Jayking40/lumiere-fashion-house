About the DOM

The Document Object Model (DOM) is a programming interface for web documents. It represents the page as a tree of nodes so that programs — primarily JavaScript — can change the document structure, style, and content.

Every HTML element becomes a node in the tree. Parent elements contain child elements, forming a hierarchical structure. JavaScript uses methods like document.querySelector() and getElementById() to traverse and manipulate this tree.

On the LUMIÈRE website, JavaScript interacts with the DOM to power scroll animations, the mobile menu, modal overlays, form validation, countdown timers, product filtering, and the interactive tree diagram you see above.
